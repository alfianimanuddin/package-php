# package-php
Latihan Package PHP

requirements:
1. github
2. packagist.org
3. nama vendor/name harus sama dengan github

1. composer init
2. bikin file baru .php (bebas)
3. edit composer.json, tambahkan :
    "autoload": {
          "files": [
              "Hello.php"
          ]
      }
      
4. membuat version dari package yg kita buat
